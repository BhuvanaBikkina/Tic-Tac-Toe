# Tic Tac Toe Game (Java)

This is a simple **2-player Tic Tac Toe** game made using **Java**.  
The game is played in the console. Players take turns entering row and column values to place **X** and **O** on a 3x3 board.  
It checks for a winner (row/column/diagonal), detects draw, and allows replay.

## How to Run
```bash
javac Main.java
java Main
